Tutorial HTML y CSS

Estructura

Estructura HTML

  • Etiqueta html: Una línia que conté informació sobre la versió d'HTML

  • Etiqueta head: Indica la informació tècnica per al navegador

  • Etiqueta body: Conté el que apareixerà a la pàgina web

Estructura CSS

  • Selector: És el nexe d'unió entre el full d'estils i els documents als quals s'apliqui aquesta fulla.

  • Propietat: Pot definir propietats de regles CSS, com la font de el text, la imatge i el color de fons, les propietats d'espaiat i disseny i l'aspecte dels elements de les llistes.

  • Valor: indica el valor de la propietat

  • Separador: Separa la propietat a la valor

  • Declaració: És el que va després de l'selector

Etiquetas

Etiquetas HTML

Podem fer servir les etiquetes HTML per crear paràgrafs de text (p), inserir imatges a la nostra pàgina web (img), crear taules (table), construir llistes (ul, li), ressaltar paraules o frases en negreta (strong) i un llarg etcètera.

Etiquetas CSS

En el full d'estil arxiu. css s'escriu com volem que sigui l'estil de presentació de les etiquetes color, mida, font, vores, marges, posició. A la pàgina web s'escriu quin full d'estil volem utilitzar.

Atributs

Els atributs d'HTML són paraules especials utilitzades dins de l'etiqueta d'obertura, per controlar el comportament de l'element. Els atributs d'HTML són un modificador d'un tipus d'element d'HTML.

Reglas

Cada un dels estils que componen un full d'estils CSS. Cada regla està composta d'una part de "selectors", un símbol de "clau d'obertura" {, una altra part denominada "bloc de declaració" i finalment, un símbol de "clau de tancament "}.

Propiedades

Són les que permeten assignar el valors

color: Defineix el color que s'utilitzés HTML

font-family: Defineix la font que s'utilitzés la per a un element

font-size: Defineix la mida de el text que s'utilitzés per l'HTML

text-align: Defineix l'alineació horitzontal d'un element HTML

Valors

Valors HTML

Són les propietats que se li poden assignar als elements. Per exemple, a html se li pot assignar un atribut per indicar en quin idioma està escrit el document (lang) i podem assignar com a valor un codi d'idioma "és" per l'espanyol.

Valors CSS

Un valor en CSS és una forma de definir una col·lecció de subvalores.
